# Mesa de Ayuda - Documentación

Bienvenido a la documentación técnica del sistema de Mesa de Ayuda.

## ¿Qué es este proyecto?

Sistema de gestión de tickets de soporte técnico desarrollado en Spring Boot.

## Características principales

- ✅ Gestión completa de tickets (CRUD)
- ✅ Asignación automática a técnicos
- ✅ Dashboard de métricas
- ✅ API REST documentada

## Tecnologías

| Capa | Tecnología |
|------|------------|
| Backend | Spring Boot 3.x, Java 17 |
| Base de datos | PostgreSQL |
| Frontend | Angular |
